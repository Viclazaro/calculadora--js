## ➢ Sobre o Projeto

A **Calculadora em JavaScript** é um projeto criado para reforçar conceitos fundamentais da programação de forma prática.

O programa realiza operações matemáticas básicas utilizando:
- funções
- `switch case`
- parâmetros
- `return`

## ▶️ Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/CalculadoraJS.git
```

2. Acesse a pasta do projeto:

```bash
cd CalculadoraJS
```

3. Execute o arquivo JavaScript no terminal
ou utilize a extensão **Code Runner** no VS Code
